# キティホーク 1903 / Kitty Hawk 1903

December 17, 1903 — 10:35 AM, Kill Devil Hills, North Carolina.
The first powered, sustained, controlled human flight.

A single-page HTML5 game. Pilot the Wright Flyer down its wooden launching rail, fight the porpoising forward canard, and try to match the Wright Brothers' four flights of December 17:

- **120 ft** — Orville, Flight 1 (12 sec)
- **175 ft** — Wilbur, Flight 2
- **200 ft** — Orville, Flight 3
- **852 ft** — Wilbur, Flight 4 (59 sec)

Tap and hold to pull the elevator and climb; release to nose down. Bilingual (日本語 / English). No libraries, no build step — just open `index.html`.

Play: https://pndshch.github.io/kitty-hawk-1903/
